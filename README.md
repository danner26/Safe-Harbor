# Safe Harbor

Safe Harbor is an open-source aquarium tracker for self-hosted tank records, livestock history, water-quality measurements, charts, and mobile-friendly logging.

<!-- TODO 2c: screenshots -->

## Quickstart

1. Clone the repository.
2. Install Python 3.12 and `uv`.
3. Run `uv sync --extra dev`.
4. Copy `.env.example` to `.env` and adjust local settings.
5. Follow [the installation guide](docs/install.md).
